# Family Hub

Self-hosted application to help my family organize everyday tasks in one place. It provides shared tools for household management features while serving as a personal project.

## Features
- Shared shopping list management
- Telegram bot integration (group chats and private messages)
- PostgreSQL database
- Docker and Docker compose
- Self-hosted deployment

## Future Improvements
- Expense Tracker
- Reminders
- Sinking Funds
- CI/CD for deployments
